<img src="Assets/icon.png" align="left" /> zEncoder 
======
Simple text converting tool, created with javascript.

The library file is located at `Lib/zEnc.js` and compatible with `nodejs`.

#### Supported Codecs

R : `Ready`<br/>
B : `Built-n`

|Codec|Stat|Encoder|Decoder|
|--|--|--|--|
|zEncode|`Removed`|
|CharCode|B|B|B|
|Attbash|Ready|R|R|
|Binary|Ready|R|R|
|URI Component|B|B|B|
|Yolious Gaiser|Ready|R|R|
|Hex|Ready|R|B|
|Base64|B|B|B|
